OpenAI Settings

How to get an OpenAI API key:

  1. Go to OpenAI and create an account
  2. Navigate to the API section
  3. Create a new API key
  4. Copy and paste the key above

Google Gemini Settings

How to get a Gemini API key:

  1. Go to Google AI Studio and create an account
  2. Click on "Get API key" in the API section
  3. Create a new API key
  4. Copy and paste the key above

Note: Your API keys are stored locally on your device and are never sent to our servers.
